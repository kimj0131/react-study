1. JSX

	- JavaScript + XML/HTML
	- 자바스크립트 내부에서 HTML과 유사하게 생긴 XML을 사용하여 코드의 가독성을 올리기 위한 추가 문법(확장 문법)
	
	const btn1 = React.createElement('button', {onClick: () => {} }, '잘하셨습니다.');
	const btn2 = <button onClick={ onClick: () => {} }> 잘하셨습니다 </button>

	- JSX 문법 내부에서 자바스크립트를 사용하고 싶다면 {}로 영역을 생성한다
	- 문자열 이외의 타입을 사용하고 싶다면 {}로 감싼 후 데이터를 사용해야 한다

2. React.createElement(type, props, [children])

	- 리액트에서 원하는 컴포넌트로 요소를 생성하기 위한 메서드
	- type : 기본 HTML태그 또는 다른 리액트 컴포넌트를 넣어서 요소를 생성할 수 있다.
	- props : 해당 요소를 생성할 때 필요한 속성 값들을 자바스크립트 객체 형태로 전달한다.

	props의 예시
	{
		id : 1,
		password : 'abc1234',
		userName : '사용자1'
	}
	- children : 해당 요소 내부의 내용이 어떤것들이 있는지 전달한다
				 (텍스트 노드, 다른 컴포넌트, 다른 태그 등등)
