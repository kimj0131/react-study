00_React 시작하기.txt

1. 설치 순서
	(1) VS Code 설치
	(2) Node.js 설치
	(3) Ctrl + Shift + `로 터미널을 연 후 
	(4) npx creat-react-app [프로젝트명]
		안될경우 npm install npm -g 로 전역설치 진행
	(5) cd 프로젝트폴더명
	(6) 프로젝트 위치에서 npm start >> 서버켜짐

	** Ctrl + c : 서버 꺼짐

2. React란 무엇인가
	- 웹 페이지를 좀더 체계적으로 개발할 수 있게 되는 자바스크립트 라이브러리
	- HTML/CSS/JavaScript를 마구잡이로 사용하다보면 유지보수가 힘들어지는 문제를 해결하기 위해 만들어진 라이브러리이다
	- 페이스북을 만든 개발자들이 사용했던 라이브러리가 오픈 소스로 공개된 것이다

3. React의 특징
	- 마크 주커버그가 밀어주는 라이브러리이기 떄문에 안정감이 높아 "커뮤니티"가 많이 활성화 되어있다
	- 계속 바뀌기 때문에 끊임없이 공부해야 한다
	- 컴포넌트 단위로 웹 페이지를 조립하듯이 구성한다
	- SPA(Single Page Application)를 개발하기에 최적화 되어있는 라이브러리이다 (Rest 최적화)
	- 비교적 적은 비용으로 DOM을 교체할 수 있게끔 실제 DOM이 아닌 가상 DOM을 사용한다
